# LeaderTask Platforms

Access LeaderTask on any device, anywhere.

## Available Platforms

### Windows

Full-featured desktop application for Windows 10 and 11.

### Web

Access LeaderTask from any web browser with our responsive web interface.

### Android

Native Android app available on Google Play Store.

### iOS

Native iOS app available on the App Store.

### macOS

Native macOS application for Mac users.

## Cross-Platform Synchronization

All your data is automatically synchronized across all platforms, ensuring you always have access to your tasks and projects.

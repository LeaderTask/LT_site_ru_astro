# Платформы LeaderTask

Доступ к LeaderTask на любом устройстве, где бы вы ни находились.

## Доступные платформы

### Windows

Полнофункциональное десктопное приложение для Windows 10 и 11.

### Web

Доступ к LeaderTask из любого веб-браузера с помощью нашего адаптивного веб-интерфейса.

### Android

Нативное приложение для Android, доступное в Google Play Store.

### iOS

Нативное приложение для iOS, доступное в App Store.

### macOS

Нативное приложение для macOS для пользователей Mac.

## Кроссплатформенная синхронизация

Все ваши данные автоматически синхронизируются на всех платформах, обеспечивая постоянный доступ к вашим задачам и проектам.

Привет. Сегодня подводим итоги февраля. Этот месяц выдался особенно продуктивным для нашей команды разработчиков — вышло несколько релизов для iOS, а на Android обновили базу данных.&nbsp;

Мы подготовили видео с обзором всех обновлений этого месяца. Рекомендуем начать именно с него. Приятного просмотра!

## ****Что Нового в ЛидерТаске****&nbsp;

### **Обновления для iOS**

Для iOS вышло несколько обновлений, и часть из них была направлена на доработку канбан-досок.&nbsp;

- Добавили статистику по доскам, которая позволит оценить эффективность команды.

- Появились напоминания для карточек.

- Реализовали фильтры в досках.

- В чатах появилась возможность ответа на сообщение.

- Добавили возможность переноса всего столбца в другую доску.

- Добавили отображение фона досок, а у карточек появились обложки.

- Теперь в чате карточки указывается ее создатель.&nbsp;

- В доски можно приглашать пользователей из других организаций.&nbsp;

- При назначении вас ответственным за карточку вам придет уведомление.

- При долгом тапе на месяц в календаре появляется барабан с выбором даты/месяца.

- ЛидерТаск теперь умеет распознавать номера телефонов в названиях и описаниях карточек.&nbsp;

- В чатах появился фильтр «Показать только файлы».&nbsp;

Про канбан-доски на iOS подробно рассказывали в этом видео.

Занимались багфиксом. Исправили много багов, вот некоторых из них:

- баг с точками в календаре;

- баг, когда в навигаторе отображались разделы без задач;

- баг, когда нельзя было войти в приложение с превышенным количеством пользователей.&nbsp;

### **Обновления для Android**

Команда android-разработчиков завершила переписывание базы данных. Это значительно ускорило работу приложения. Данные теперь обрабатываются быстрее, пропали пустые списки и другие назойливые баги.&nbsp;

Для нас переписывание базы данных было одной из ключевых задач, так как от этого зависит не только работа приложения, но и скорость разработки. Как итог, релизы теперь будут чаще, и мы сможем сконцентрироваться на добавлении канбан-досок и других функций.

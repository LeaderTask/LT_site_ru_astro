Диалог настройки служит для установки пользователем индивидуальных параметров работы.

![Настройка планировщика](https://www.leadertask.ru/images/options1.gif)

- **Язык&nbsp;**– Выбор языка интерфейса программы

- **Загружаться вместе в Windows**&nbsp;– установите отметку если хотите чтобы программа автоматически загружалась вместе с ОС Windows

- **Закрывать в трей**&nbsp;– установите отметку если хотите чтобы при клике на закрытие программы- программа сворачивалась в область системного трея.

- **Перенос по словам**&nbsp;– установите отметку если хотите чтобы задачи имеющие длинное наименование -&nbsp; разбивались по строкам.

- **Формат даты&nbsp;**– установка формата даты для отображения в дереве задач

- **Делать резервную копию при выходе?**&nbsp;– установите да если желаете делать резервную копию вашей базы данных. Мы настоятельно рекомендуем установить это значение.

- **В папку&nbsp;**– установите папку для хранения резервных копий. По умолчанию стоит папка backup находящаяся в текущей папке LeaderTask.

![Настройка планировщика - Календарь](https://www.leadertask.ru/images/options2.gif)

- **Размерность шкалы&nbsp;**– установка размерности шкалы времени в календаре

- **Рабочая неделя**&nbsp;– установка рабочих дней. Установите значения у рабочих дней. Начало недели – установите начальный день недели. Начало дня – время начала рабочего дня, Конец&nbsp; дня – время завершения рабочего дня.

- **Сжимать дни&nbsp;**– установите это значение если хотите отображать СБ и ВС в сжатом виде.

![Настройка планировщика - Оформление](https://www.leadertask.ru/images/options3.gif)

- **Стиль таблицы задач&nbsp;**– выберите стиль для таблицы дерева задач.

- **Стиль линий**&nbsp;– выберите стиль&nbsp; линий для таблицы дерева задач.

- **Рисовать вертикальные линии**&nbsp;– установите значение рисовать или нет вертикальные линии в дереве задач

- **Процент смешивания с таблицей**&nbsp;– установите значение процента смешивания цвета с таблицей

- **Процент смешивания с линиями**&nbsp;– установите значение процента смешивания цвета с линиями

![Настройка планировщика - Заметки](https://www.leadertask.ru/images/options4.gif)

- **Шрифт заметок по умолчанию**&nbsp;– установите шрифт и размер шрифта в заметках по умолчанию.

- **Показывать панель наименования в редакторе заметки**&nbsp;– установите если требуется.

- **Автоматически вставлять данные из буфера обмена в пустую заметку**&nbsp;– установите если требуется

![Настройка планировщика - Задачи](https://www.leadertask.ru/images/options5.gif)

- **Шрифт задач**&nbsp;– установите шрифт задач для отображения всех задач в дереве.

- **При выполнении задачи переносить дату завершения на сегодня**&nbsp;– установите если требуется.

- **Создание задачи по мере набора**&nbsp;– установите значение если желаете вводить задачи путем непосредственного набора текста задачи в дереве задач.

![Настройка планировщика - Предупреждения](https://www.leadertask.ru/images/options6.gif)

Установите отметки у тех предупреждений которые хотите&nbsp; видеть при каждом событии.

![Настройка планировщика - Дополнительно](https://www.leadertask.ru/images/options7.gif)

- **Отображать суммарное количество задач в навигаторе**&nbsp;– опция работает&nbsp; только после перезагрузки программы. В навигаторе будут отображаться кол-во задач в каждом элементе.

- **Оттенять столбец по которому ведется сортировка**&nbsp;–установите если требуется.

- **Игнорировать раскраску задач при печати**&nbsp;– мы рекомендуем установить это значение если у вас ЧБ принтер

- **Выделять жирным дни в которых есть задачи**&nbsp;– опция относится к календарю периодов в навигаторе.

- **Не менять дату завершения при переносе в календарь**&nbsp;–установите если требуется.

- **Отображать номер недели**&nbsp;-&nbsp; опция относится к календарю периодов в навигаторе.

- **При перетаскивании задачи на другой проект/категори****ю/контакт менять родительскую задачу**&nbsp;– установите если требуется.

- Крупное обновление задач и исправление ошибок

- Добавили вызов меню по клику правой кнопки мыши на карточку

- Добавили переключение карточек через стрелочки внутри одной колонки

- Добавили перенос карточки в календарь, в результате создается задача с ссылкой на карточку

- При поручении задачи, теперь исполнителю приходит уведомление в Telegram (функция работает с плагином "Телеграм")

WindowsAndroid&nbsp;iOS (iPhone и iPad)&nbsp;Веб (браузерная версия)&nbsp;MacOS&nbsp;

## Как это работает

Создайте интеллект-карту в программе&nbsp;[MindManager](http://www.mindjet.com/ru/), сохраните карту в файл .xmmap, затем импортируйте в ЛидерТаск. Автоматически будет создан проект. Внутри него вы найдёте дерево задач, повторяющее структуру интеллект-карты.

Доступно также и обратное действие. В ЛидерТаске выберите любой проект, экспортируйте его в интеллект-карту (файл .xmmap), затем импортируйте карту в программу MindManager.

Внимание!&nbsp;Импорт и экспорт интеллект-карт возможен только в интерфейсе ЛидерТаска для Windows.

## Для кого будет полезно

Модуль импорта и экспорта интеллект-карт будет полезен для всех, кто хочет создавать карты мыслей и составлять по ним списки задач.

## Возможности

- Импорт интеллект-карт формата .xmmap

- Экспорт проектов в интеллект-карты формата .xmmap

## Как это работает

Модуль встраивается в интерфейс ЛидерТаска для Windows. Следовательно, работа со SMART Ассистентом производится внутри ЛидерТаска.

## Что такое «SMART»

SMART - это методика, которая позволяет формулировать цели таким образом, чтобы гарантированно их достигать. Формулирование целей производится с использованием 5 специальных правил.

## Для кого будет полезно

Модуль будет полезен для всех, кто хочет быстро достигать целей, затрачивая минимум усилий.

## Возможности

- Создание целей

- Исправление формулировок целей в соответствии с 5 правилами SMART

- Хранение списка целей

- Поддержание одинакового списка целей в программе и ЛидерТаске

## Можно ли использовать программу отдельно от ЛидерТаска

Нет, нельзя. Модуль «SMART Ассистент» встраивается в интерфейс ЛидерТаска для Windows. Следовательно, работа с целями производится внутри ЛидерТаска.

## На каких платформах можно работать в программе

Модуль предназначен только для платформы Windows.

![Новое обновление Web-версии ЛидерТаск](https://www.leadertask.ru/wp-content/uploads/2023/10/Screenshot_9-2.png)

- Добавлена функция "Повтор" у задачи!

- Добавлен импорт канбан-досок из Notion!

- Перемещение задач в списке с помощью Ctrl+Вверх/Ctrl+Вниз

- Добавили возможность открыть ежедневник на весь экран

- Добавили эмодзи для чатов задач и карточек

- Теперь вы можете видеть онлайн на задачах (чтобы понимать кто кроме вас просматривает задачу прямо сейчас)

## Поддерживаемые платформы

WindowsAndroid&nbsp;iOS (iPhone и iPad)&nbsp;Веб (браузерная версия)&nbsp;MacOS&nbsp;

## Как это работает

Диаграмма Гантта встраивается в интерфейс ЛидерТаска для Windows. Следовательно работа с диаграммой производится внутри ЛидерТаска.

## Для кого будет полезно

Модуль будет полезен для менеджеров проектов, а также для руководителей всех уровней.

## Возможности

- Отображение задач проекта на временной шкале

- Перемещение задачи по временной шкале

- Изменение длительности задачи

- Добавление новых задач в проект

- Удаление задач

- Открытие окна свойств задачи

- Перемещение по временной шкале влево и вправо с помощью драг-н-дропа (перетаскивания)

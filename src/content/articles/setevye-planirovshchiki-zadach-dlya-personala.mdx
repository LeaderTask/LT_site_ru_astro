Как организовать работу сотрудников по сети через компьютер? Как систематизировать контроль выполненных задач? На все эти вопросы ответит&nbsp;[планировщик](https://www.leadertask.ru/blog/planirovshhiki-zadach)&nbsp;LeaderTask.

## Для самостоятельной работы сотрудников

- Ежедневное расписание (шкала времени)

- Утреннее уведомление о предстоящих задачах

- Список задач на сегодня

- Установка времени для выполнения задач

- Установка стандартных и регулярных уведомлений (для повторяющихся задач)

![Самостоятельная работа сотрудников в ЛидерТаск](https://www.leadertask.ru/ckfinder/userfiles/files/%D0%94%D0%B8%D0%B7%D0%B0%D0%B9%D0%BD%20%D0%B1%D0%B5%D0%B7%20%D0%BD%D0%B0%D0%B7%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F%20(5)(4).png)

## Для работы с задачами

- Создание задач и подзадач

- Создание проектов, подпроектов

- Перемещение задач в проекты

- Перенос задач на другую дату

## Для важных задач

- Добавление цветных меток с любым названием

- Обозначение задач цветом

- Задачи "В фокусе"

![Обозначение важных задач в ЛидерТаск](https://www.leadertask.ru/ckfinder/userfiles/files/%D0%94%D0%B8%D0%B7%D0%B0%D0%B9%D0%BD%20%D0%B1%D0%B5%D0%B7%20%D0%BD%D0%B0%D0%B7%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F%20(1)(3).png)

## Для совместной работы

- Поручение, перепоручение задач

- Отправка файлов (изображения, ссылки, документы)

- Добавление заметок, чек-листов

- Обсуждение проектов

- Предоставление доступа к задачам

- Всплывающие уведомления

![Обсуждение задач в ЛидерТаск](https://www.leadertask.ru/ckfinder/userfiles/files/%D0%94%D0%B8%D0%B7%D0%B0%D0%B9%D0%BD%20%D0%B1%D0%B5%D0%B7%20%D0%BD%D0%B0%D0%B7%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F%20(6)(4).png)

## Для руководителей

- Задачи "Поручено мной"

- Просмотр всех задач сотрудников

- Список отделов, сотрудников

- Отправка задач на доработку

- Внесение корректировок в задания

- Архив завершенных задач

## Задачи для сотрудников

- Задачи "Поручено мне"

- Установка статуса для задач: "Отложено", "Отменено", "В работе", "Готово к сдаче".

При установке статуса "В работе", на главном экране появится вкладка со списком задач, над которыми сотрудник сейчас работает.

## Быстрые команды в ЛидерТаск

- Создание задач прямо из браузера

- Создание задач из электронных писем

- Доступы к задачам, проектам и файлам по ссылкам

## Информация при работе с задачами

- Поиск данных в программе

- Сортировка и фильтрация задач (по дате, меткам, цветам, проектам, по столбцам, по возрастанию и др)

- Просмотр задач на несколько дней (выбор любого периода)

- Печать данных из программы

Поручайте задачи сотрудникам, принимайте задания и устанавливайте статус "завершено" или отправляйте задачи на доработку. Наблюдайте за процессами компании со своего компьютера, ноутбука или другого устройства. Программа, с большим функционалом, которая поможет управлять задачами легко и интересно.

## Органайзеры делятся на:&nbsp;

- Органайзеры для личного использования

- Органайзер для компании и делегирования задач

- Органайзеры&nbsp;для ведения и хранения данных

- Органайзеры - программы для ведения продаж

## Возможности

- Добавление комментариев, заметок, чек-листов

- Добавление файлов, изображений, ссылок (отправка)

- Доступы к задачам (общие и индивидуальные)

[calenar_functional]

- Добавление цветных меток к заданиям, присвоение любого цвета для задач

- Статус выполнения задачи

- Задачи "Поручено мне", задачи "Поручено мной"

- Отправка задач на доработку

## Органайзер LeaderTask для ведения и хранения данных:

- Хранение файлов, изображений, заметок в программе внутри задач

- Доступ к данным (задачам, поручениям, проектам и т.д.) даже без&nbsp;интернета

- Синхронизация данных

- Архив завершенных задач

- Выгрузка и загрузка данных из программы

## Органайзеры - программы для ведения продаж:

- Составление плана продаж на будущее,&nbsp;на любой период

- Создание проектов с любым названием, например, "Продажи март"

- Добавление меток: внимание, срочно, др. (любое название для меток)

- Регулярные напоминания (каждый день, будни, месяц, год)

- Преобразование писем в задачи

- [Планирование в любых сферах](https://www.leadertask.ru/blog/planirovanie)

- Постановка целей даже без интернета

## Для кого будет полезно

Модуль будет полезен для работников отдела кадров, руководителей проектов, начальников отделов, генеральных директоров, то есть для тех, кто управляет людьми.

## Видеообзор

## Можно ли использовать программу отдельно от ЛидерТаска

Нет, нельзя. Модуль «Отпуска сотрудников» встраивается в интерфейс ЛидерТаска для Windows. Следовательно, работа с отпусками производится внутри ЛидерТаска.

## На каких платформах можно работать в программе

Модуль предназначен только для платформы Windows.

## Нужно ли платить за новые версии

Нет, не нужно. Новые версии программы «Отпуска сотрудников» вы получите бесплатно!

![Создание задач в ЛидерТаск из писем ](https://www.leadertask.ru/ckfinder/userfiles/files/27.png)

Создавайте задачи из электронных писем со всем ее содержанием.&nbsp;Всего два клика и письмо превращается в задачу. Что нужно сделать, чтобы создать задачу из электронной почты:

- Выбрать нужное письмо

- Отправить по адресу:&nbsp;[x@leadertask.com](mailto:x@leadertask.com)

![Формирование задач из писем](https://www.leadertask.ru/ckfinder/userfiles/files/%D0%BF%D0%BE%D1%87%D1%82%D0%B0-min.png)

Письмо моментально появится в программе&nbsp;ЛидерТаск в виде задачи, в разделе "Неразобранное".

- тема письма отобразит название задачи

- содержание письма появится в примечании

- все файлы отправятся в саму задачу

Письмо&nbsp;необходимо отправлять с адреса, который привязан в LeaderTask.&nbsp;

Создавайте задачи из писем в неограниченном количестве! Это так просто!

А если вам нужно создавать быстрые задачи прямо из браузера, используйте расширение&nbsp;LeaderTask. Для этого нужно выбрать один из вариантов:&nbsp;"Создать задачу" или "Скриншот" и задача улетает в программу!

![Быстрое создание задач из писем](https://www.leadertask.ru/ckfinder/userfiles/files/%D0%B2%D0%B8%D0%B4%D0%B6%D0%B5%D1%82%D1%8B-min.png)

Оцените скорость создания задач из писем!

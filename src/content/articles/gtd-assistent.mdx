Удобный модуль-ассистент поможет вам быстро разобрать входящие задачи, используя метод GTD.

![GIF GTD Ассистент](https://www.leadertask.ru/images/507e43c7ba8df247447cc0b837452d2d.gif)

## Поддерживаемые платформы

WindowsAndroid&nbsp;iOS (iPhone и iPad)&nbsp;Веб (браузерная версия)&nbsp;MacOS&nbsp;

## Как это работает

Модуль встраивается в интерфейс ЛидерТаск для Windows. Следовательно, работа с «GTD Ассистентом» производится внутри ЛидерТаск.

## Суть методики «[GTD](https://www.leadertask.ru/blog/gtd)»

Суть методики повышения личной эффективности «GTD» (Getting Things Done - "Доведение дел до завершения") основывается на очень простом принципе: избавить свою голову от всей лишней информации. В нашем случае - от запоминания каких-либо задач.

## Для кого будет полезно

Модуль будет полезен для всех, кто хочет работать по методике GTD.

## Возможности

- Сортировка всех задач по трем принципам: "Задача", "Заметка", "Мусор"

- Назначение задаче определенной даты выполнения

- Поручение задачи кому-то из коллег

- Возможность отложить задачу в "Архив"

- Возможность отправить задачу в любой проект

Только для руководителей!

## Поддерживаемые платформы

WindowsAndroid&nbsp;iOS (iPhone и iPad)&nbsp;Веб (браузерная версия)&nbsp;MacOS&nbsp;

## Как это работает

Отчёт по сотрудникам встраивается в интерфейс ЛидерТаска для Windows. Следовательно, создание отчёта производится внутри ЛидерТаска.

## Для кого будет полезно

Отчёт будет полезен для тех сотрудников, у кого есть подчинённые: руководителей компаний, директоров отделов, менеджеров проектов.

## Возможности

- Построение рейтинга сотрудников по количеству завершённых поручений

- Построение рейтинга сотрудников по количеству завершённых важных поручений

- Построение рейтинга сотрудников по количеству просроченных поручений

- Построение диаграммы загруженности сотрудников

- Построение рейтинга сотрудников по поручениям, привязанным ко времени

- Отображение самых обсуждаемых поручений

- Отображение самых долго ожидаемых поручений

- Построение общего рейтинга сотрудников

- Выявление самого результативного сотрудника

- Выявление самого общительного сотрудника

- Выявление самого загруженного сотрудника (по часам)

- Выбор произвольного периода для построения отчёта

- Возможность построить отчёт по конкретным сотрудникам, а также по группам сотрудников
